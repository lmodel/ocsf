    Class: IncidentFinding

    An Incident Finding reports the creation, update, or closure of security

    incidents as a result of detections and/or analytics.
    Note:

    Incident Finding implicitly includes the

    incident profile and it should be added to the

    metadata.profiles[] array.

    URI: ocsf:IncidentFinding

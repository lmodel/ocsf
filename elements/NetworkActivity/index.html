      denotes that the event source has arbitrarily assigned one peer to <code>src_endpoint</code>

      responder are not being asserted. This can occur, for example, when the event
      source is a network appliance that has not observed the initiation of a given
      connection. In the absence of this attribute, interpretation of the initiator
      and responder is implementation-specific.
    deprecated: Use <code>initiator_id</code> instead, which provides richer semantics
      for bi-flow and asymmetric flow scenarios.

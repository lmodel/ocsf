    Class: Advisory

    The Advisory object represents publicly disclosed cybersecurity vulnerabilities

    defined in a Security advisory. e.g. Microsoft KB Article,

    Apple Security Advisory, or a GitHub Security Advisory

    (GHSA)

    URI: ocsf:Advisory
